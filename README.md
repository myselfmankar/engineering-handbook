A professional-grade, automated **System Design Wiki** and engineering knowledge base. Built for high-scale learning and distilled technical mastery.

---

## 🚀 Key Features

- **Recursive Documizer Engine**: Zero-maintenance content management. Add a folder, and the site builds itself (Sidebar + Homepage cards).
- **Minimalist SaaS Aesthetic**: High-contrast grayscale theme designed for focus and engineering maturity.
- **Git-Driven Authorship**: Automatic tracking of contributors on every page via Git metadata.
- **Architecture Visualization**: Built-in support for **Mermaid.js** diagrams within Markdown.

---

## 🛠️ Tech Stack

- **Core**: [Docusaurus v3](https://docusaurus.io/) (React + TypeScript)
- **Styling**: Infima (Custom Grayscale Theme)
- **Automation**: Node.js Folder-Scanning Engine
- **Search**: `@easyops-cn/docusaurus-search-local`
- **deployment**: GitHub Actions + GitHub Pages

## 🤝 Contributing

This project is a collaborative effort. If you are a friend or fellow engineer looking to contribute:
1. **Fork** the repository.
2. **Add** a folder in `/docs` with a `_category_.json`.
3. **Submit** a Pull Request!

Check out our [Contribution Guide](https://myselfmankar.github.io/docs/handbook/contribute) for more details.
